Angular Directives: Deep Study

Directives are classes that add additional behavior to elements in your applications. With Angular's built-in directives, you can manage forms, lists, styles, and what users see.

1. Structural Directives

Structural directives are responsible for HTML layout. They shape or reshape the DOM's structure, typically by adding, removing, or manipulating elements. They are preceded by an asterisk (*).

*ngIf

Definition: *ngIf conditionally includes a template based on the value of an expression. If the expression evaluates to true, the element is added to the DOM; otherwise, it is removed.

Code Example: 
<p *ngIf="isVisible">This paragraph is visible!</p>
Output in Browser:

This paragraph is visible!

*ngFor

Definition: *ngFor is a repeater directive that you can use to display a list of items. It loops through a given array and instantiates a template for each item.

Code Example: 
<li *ngFor="let student of students">{{ '{{' }} student {{ '}}' }}</li>
Output in Browser:
  • {{ student }}

ngSwitch, *ngSwitchCase

Definition: ngSwitch is like a JavaScript switch statement. It displays one element from among several possible elements, based on a switch condition.

Code Example: 
<div [ngSwitch]="selectedSubject">
  <p *ngSwitchCase="'Math'">Mathematics</p>
  <p *ngSwitchDefault>No Subject Selected</p>
</div>
Output in Browser:

Subject: Mathematics 📐

Subject: Science 🔬

Subject: English 📚

Unknown Subject

2. Attribute Directives

Attribute directives change the appearance or behavior of an element, component, or another directive.

[ngStyle]

Definition: The [ngStyle] directive lets you set a given DOM element's style properties dynamically within a component.

Code Example: 
<p [ngStyle]="{ 'color': isHighlight ? 'blue' : 'black' }">Styled Text</p>
Output in Browser:
This text changes style based on the component's state!

[ngClass]

Definition: The [ngClass] directive allows you to add or remove CSS classes on an HTML element.

Code Example: 
<div [ngClass]="{ 'active-box': isActive, 'inactive-box': !isActive }">Box</div>
Output in Browser:
{{ isActive ? 'Active State' : 'Inactive State' }}
